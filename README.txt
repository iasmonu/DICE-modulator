DICE-modulator
---------------------
This dice modulator produces random faces for random numbers inn the dice

Getting Started
---------------
There 1 python file and one algorithm file :
dice_simulator.py: this python file contains code for the dice simulator 
		   which is a basic program to generate random faces of dice.

alogrithm file: This is a word file explaining each step involved in creating
                the program


Prerequisites
-------------
> pyhton installed 
> if not installed go to the link: https://www.python.org/downloads/

> Atom.io: Editor atom if installed and can be used to run the python file    


Running the tests:
------------------
Access the dice_simulator.py either by python idle or the atom editor to run the file
